# TP1 : Héritage, redéfinition, Classes abstraites, Interfaces et polymorphisme .

Exercices pratiques de Programmation Orientée Objet (Héritage, Polymorphisme).

## Structure du projet

* **ex1** : Gestion de bibliothèque (Classes `Personne`, `Adherent`, `Livre`, etc.).
* **ex2** : Gestion de véhicules (Classes `Vehicule`, `Voiture`, `Moto`, `Avion`).

## Comment lancer le projet

Ouvrez un terminal à la racine du dossier et lancez les commandes suivantes :

```bash
javac ex1/*.java
java ex1.Program
````
------
```bash
javac ex2/*.java
java ex2.Program
```
