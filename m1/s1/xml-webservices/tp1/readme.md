# Activité pratique N°1 - XML DTD XSD XPath XSL SVG

> [showcase](https://mmaous.github.io/ms-iibdcc/m1/s1/xml-webservices/tp1/)

Pour une démonstration complète de toutes les étapes, naviguez à .

Un exercice de classe sur les technologies XML, DTD, XSD, et les transformations XSLT, basé sur le cours de M. Youssfi.

Ce projet démontre le cycle de vie complet d'un document XML :

- Définition d'une structure de données (températures des villes).
- Validation de la structure via DTD et Schéma XML (XSD).
- Transformation des données XML en une page web HTML.
- Transformation des données XML en une visualisation de données (Histogramme SVG animé).

## Contenu du projet

- meteo.xml: Le document XML de base contenant les données de température.

- meteo.dtd: La Définition de Type de Document (DTD) pour la validation.

- meteo.xsd: Le Schéma XML (XSD) pour une validation plus moderne et stricte.

- meteo_to_html.xsl: Feuille de style XSLT pour transformer le XML en une page HTML simple.

- meteo_to_svg.xsl: Feuille de style XSLT pour transformer le XML en un histogramme SVG animé.

- représentation graphique de l’arbre XML.png: L'arbre structurel du document XML.

- meteo.html: Le fichier HTML généré par la transformation XSLT.

- meteo.svg: Le fichier SVG généré par la transformation XSLT.
